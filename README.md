# Bittree Clone

A **Link in Bio** web app inspired by Linktree, built with **Next.js**, **React**, and **MongoDB**. Users can claim a handle, add multiple links, and customize their profile with a picture and description.

## Features
- Claim a unique handle
- Add multiple links with custom text
- Profile picture and description
- Dynamic routing per handle (`/generate?handle=<handle>`)
- Responsive UI with Tailwind CSS
- Persistent storage with MongoDB
- Success/error notifications with react-toastify

## Tech Stack
- **Frontend:** Next.js 13, React, Tailwind CSS
- **Backend:** Next.js API Routes
- **Database:** MongoDB
